Registro de proveedor

Alta corta, validación clara y perfil listo para revisión.

Este flujo está pensado para proveedores que necesitan registrar su cuenta en pocos pasos, mantener la misma estética institucional y dejar un perfil completo para la aprobación manual.

4 pasos Acceso, base del perfil, datos específicos y confirmación final.
Estado pendiente La cuenta se activa solo cuando administración valida la solicitud.
Datos reales El formulario usa catálogos del backend para evitar valores inventados.
